INDHOLD - SPØRGSMÅL (12 TOTAL)
================================================================================

LET (4 stk × 1pt = 4pt max per rigtigt)
──────────────────────────────────────────
1. Hvad er normal spænding σ?
   → σ = P/A (kraft ÷ areal)

2. Hvad er normal tøjning ε?
   → ε = δ/L (relativ forlængelse)

3. Hookes lov i normal spænding?
   → σ = E·ε

4. Centroide-betingelse for aksial last?
   → Kraft gennem centroiden → uniform σ = P/A

───────────────────────────────────────────

MIDDEL (5 stk × 2pt = 10pt max per rigtigt)
──────────────────────────────────────────
5. Lateral tøjning ved aksial træk (E=200 GPa, ν=0,3, ε=0,45×10⁻³)?
   → ε_lat = −0,135×10⁻³

6. Forskydningsspænding i double shear bolt (P=2kN, A=201mm²)?
   → τ ≈ 10 MPa

7. Lejespænding (bearing stress) formel?
   → σ_b = F_b/(t·d_b)

8. Hookes lov i forskydning?
   → τ = G·γ

9. Moment-ligevægt på element?
   → τ₁ = τ₂

───────────────────────────────────────────

SVÆR (3 stk × 3pt = 9pt max per rigtigt)
──────────────────────────────────────────
10. Lateral tøjning fra aksial (L=10cm, d=1cm, P=7kN)?
    → Δd ≈ −1,3 μm

11. G-modul fra E og ν (E=3 MPa, ν=0,49)?
    → G ≈ 1,0 MPa

12. Normalspænding på 45° snit under aksial træk?
    → σ₄₅ = σ_x/2

───────────────────────────────────────────

TOTAL MULIGT: 4×1 + 5×2 + 3×3 = 23 point
              (før combo-multiplikation)

================================================================================
KNOWN ISSUES & WORKAROUNDS
================================================================================

[MINOR] KaTeX not loaded:
  • Vendor/katex files missing
  • Workaround: Fallback to plain HTML rendering (works fine)
  • Fix: Run copy_vendor.ps1

[MINOR] Fonts render as serif (system fallback):
  • Vendor/fonts directory missing
  • Workaround: Quiz still fully functional, just different font
  • Fix: Run copy_vendor.ps1

[INFO] LocalStorage quota exceeded:
  • Unlikely on this quiz (~2–5 KB per score)
  • Workaround: Clear browser cache, restart
  • Future: Implement cloud sync

[FUTURE] Long questions break on mobile:
  • CSS is responsive, but UX could be improved
  • Future: Add max-width to .qtext on mobile
