M-5 · Stål, rustfrit stål, støbejern og aluminium

Ét-sidet overblik til 12-taller  ·  Juni 2026

1 Spørgsmålet — hvad skal besvares?

  • a) Definér stål + forskel på ulegeret, legeret og rustfrit stål.
  • b) De vigtigste legeringselementer i rustfrit stål: Cr, Ni, Mo, N.
  • c) De vigtigste typer rustfrit stål: ferritisk, austenitisk, martensitisk, duplex.
  • d) Overblik over støbejern + forskelle i mikrostruktur.
  • e) Aluminiumlegeringer: serier 1xxx–7xxx + modningshærdning.
Kaffe-analogi: ulegeret = sort kaffe (Fe + lidt C) · legeret = kaffe m. mælk/sukker (Cr, Ni, Mo) · rustfrit = kaffe i ståltermos (beskyttet, >10,5 % Cr).

Definition: stål = jern med <2,14 % C. Mest anvendte metalgruppe — bedste balance af styrke, duktilitet, bearbejdelighed og pris.

3 Rustfri ståltyper

Nøgle: krystalstrukturen styrer egenskaberne — FCC (austenit) er sej og ikke-magnetisk; BCC (ferrit) er billig og magnetisk; BCT (martensit) er hård.

4 Passivering & legeringselementer

Cr > 10,5 %  →  Cr₂O₃-passivfilm (2–3 nm, selvhelende)
Cr = passivfilm (ferritstabil.) · Ni = FCC/austenit (sej) · Mo = pitting-modstand (316) · N = styrke + pitting (billig Ni-erstatning).

5 Støbejern — mikrostruktur (>2 % C)

Også: hvidt støbejern = cementit-netværk (hårdt/sprødt, mellemprodukt) · tempergods = hvidt jern glødet 800–1000 °C → cementit bliver til tempergrafit (↑ duktilitet).

Modningshærdning (2xxx/6xxx/7xxx): ① opløsningsglødning ~500 °C → ② vandafkøling (overmættet) → ③ modning (intermetalliske udfældninger blokerer dislokationer). Al 6061-T6 ≈ 310 MPa vs. ~124 MPa blød.

7 Modningshærdnings-forløb

Vendepunkt: de finfordelte θ′-udfældninger låser dislokationer — for lang modning (overaldring) gør udfældningerne grove igen → styrken falder.

8 Korte svar — klar til eksamen

a) Stål + typer?Stål = Fe m. <2,14 % C. Ulegeret (Fe+C), legeret (+Cr/Ni/Mo), rustfrit (>10,5 % Cr → passivfilm).
b) Legeringselementer?Cr = passivfilm · Ni = FCC/sejhed · Mo = pitting-modstand · N = styrke + billig Ni-erstatning.
c) Rustfri typer?Ferritisk (BCC, billig/magnetisk), austenitisk (FCC, sej, 18/8), martensitisk (BCT, hård), duplex (FCC+BCC, stærk/chloridfast).
d) Støbejern?>2 % C. Gråt (flager, sprødt), hvidt (cementit, hårdt), S-G (nodler, sejt), tempergods (glødet → grafit).
e) Aluminium?Let (2,7 g/cm³), korrosionsfast. Serier 1–7xxx; 2/6/7xxx modningshærdes (glød → quench → modning).

9 Huskeregler & referencer

Hurtige huskeregler:
  • 10,5 % Cr = grænsen for "rustfrit" — lær tallet udenad.
  • 18/8 = AISI 304 (18 % Cr, 8 % Ni) = den klassiske austenit.
  • 316 = 304 + 2–3 % Mo → bedre pitting (havvand). 316L = low-C → undgår sensibilisering ved svejsning.
  • 2/6/7xxx = de modningshærdbare Al-serier.
